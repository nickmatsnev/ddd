This snippet will help you create a full-width block inside a parent that has fixed width (e.g. content wrapper and full-width image inside it).

## CSS

{{< code-header name="style.css" >}}
```CSS
.full {
  width: 100vw;
  margin-left: 50%;
  transform: translateX(-50%)
}
```
